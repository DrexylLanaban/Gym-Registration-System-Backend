# IPT Final Project Backend (REST API)

Node.js + Express + **MySQL** (`mysql2`). Typical setup: **API on Render** (or your PC) + **MySQL on InfinityFree**

## InfinityFree MySQL 



| Variable | Example |
|----------|---------|
| `DB_HOST` | `sql103.infinityfree.com` |
| `DB_PORT` | `3306` |
| `DB_USER` | InfinityFree MySQL user |
| `DB_PASSWORD` |  panel password |
| `DB_NAME` | exact name from the panel |
